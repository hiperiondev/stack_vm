Opcodes

Opcodes

op format: iiOOOOOO

  • i: use indirect register in instruction

    • 00: nothing

    • 01: indirect, not change register

    • 10: indirect, increment register

    • 11: indirect, decrement register

  • O: opcode

NOTE: @arg -> support indirect
